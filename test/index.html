Compare Objects

compares two objects recursively for equality
Try your own comparisons!

Example 1: Objects are not equal

apples.b and oranges.b are not equal 
            
                
               const apples = {
                 a: 'apple',
                 b: 'orange',
               };
               const oranges = {
                 a: 'apple',
                 b: 'apple',
               };
               compareObjects(apples, oranges);

Example 2: Objects are equal

              
                  
                const aSquad = {
                    a: 'bob',
                    b: 'shaker',
                  };
                  const bSquad = {
                    a: 'bob',
                    b: 'shaker',
                  };
                
                 compareObjects(aSquad, bSquad);

Example 3: Objects are equivalent

                
                    
                    const integers = {
                        a: 1,
                        b: 2,
                      };
                      const strings = {
                        a: '1',
                        b: '2',
                      };
                    compareObjects(integers, strings);
